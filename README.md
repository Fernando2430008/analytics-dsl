# Analytics DSL
Analytics DSL es un lenguaje de dominio especifico desarrollado en python para definir flujos de análisis de datos, incluyendo fuentes de datos, preprocesamiento y configuración de modelos de aprendizaje automático.

## Caracteristicas actuales
- Declaración de fuentes de datos con `datasource`
- Reglas de preprocesamiento con `preprocess`
- Declaración de modelos de aprendizaje con `learner`
- Análisis léxico y sintáctico usando PLY
- Validación semántica básica
